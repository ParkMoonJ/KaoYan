# 操作系统

## Week 1

### 1.1 操作系统的基本概念

#### 学习建议

- 多道操作系统四个特征：并发、共享、虚拟、异步，并发和共享是基本特征
- 并发：宏观上多道同时执行，微观上分时执行  
  并行：同一时刻多道同时执行
